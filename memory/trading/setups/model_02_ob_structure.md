## üìä Long Setup Step-by-Step

### Step 1: Identify HTF Trend
```
Daily Chart Analysis:
‚ñ° Series of HH and HL = Bullish
‚ñ° Recent BOS confirmed
‚ñ° Price above mean threshold
‚ñ° Trend is established

If bullish ‚Üí Look for bullish OBs
If bearish ‚Üí Look for bearish OBs
```

### Step 2: Mark Fresh Order Block
```
Find the last bearish candle before a strong bullish impulse:

Example:
Candle 1: Bearish, closes at 1.0750
Candle 2: Massive bullish, opens 1.0750, closes 1.0820
Candle 3-5: Continuation up

Bullish OB:
High: 1.0755
Low:  1.0745
Zone: 1.0745 - 1.0755

Key: This OB has NEVER been tested since formed
```

### Step 3: Wait for Price to Approach
```
Monitor price action:
‚ñ° Price making new highs
‚ñ° Eventually pulls back
‚ñ° Approaching OB zone
‚ñ° Don't enter yet - wait for structure

Patience is critical here!
```

### Step 4: Drop to LTF for Entry
```
Switch to 15M or 5M chart:
‚ñ° Price entering OB zone
‚ñ° Watch for structure to form
‚ñ° Look for HL to develop
‚ñ° Wait for micro BOS

Structure confirmation options:
1. Price touches OB, forms HL, breaks micro high
2. Bullish engulfing at OB
3. Pin bar rejection at OB
4. BOS on 5M timeframe
```

### Step 5: Confirm LTF Structure
```
Micro structure example:

Price at OB: 1.0750
Forms low: 1.0748 (within OB)
Bounces: 1.0755
Breaks: 1.0760 (micro BOS)

CONFIRMED! Enter on break of micro structure.
```

### Step 6: Execute Trade
```
Entry: 1.0760 (on micro BOS)
Stop: 1.0740 (below OB low)
Target 1: 1:2 R:R (1.0800)
Target 2: 1:3 R:R (1.0820)
Target 3: Next HTF liquidity

Risk: 20 pips
Reward: 40-60 pips
```

## üìä Short Setup Step-by-Step

3. **Fresh OB:** Never tested since formed

7. **Execute:** Short with stop above OB

## üé≠ OB Types and Entries

### Type 1: Fresh OB Entry
```
The OB has never been touched.

Entry: On first test
Stop: Beyond OB
Probability: Highest

Why it works:
- Institutional orders still there
- Unfilled demand/supply
- Clean price action
```

### Type 2: Mitigation Block Entry
```
OB was tested but held.

Entry: On second test
Stop: Beyond test wick
Probability: Medium-High

Why it works:
- OB already proved support/resistance
- Still fresh enough
- Traders remember the level
```

### Type 3: Breaker Block Entry
```
OB was broken, now acts as opposite type.

Example:
1. Bullish OB breaks down
2. Now acts as resistance
3. Wait for retest from below
4. Enter short

Stop: Beyond break point
Probability: Medium

Why it works:
- Polarity flip
- Trapped traders
- Momentum with break
```

## üìà Trade Management

### Entry Options

**Aggressive (Earlier):**
- Enter when price touches OB
- No structure confirmation
- Better price
- Lower win rate

**Conservative (Later):**
- Wait for micro BOS
- Structure confirmation
- Worse price
- Higher win rate

### Stop Placement

**Standard:**
- Below OB low for longs
- Above OB high for shorts
- 5-10 pip buffer

**Tight (Aggressive):**
- Below OB 50% level
- Tighter risk
- Higher chance of stop out

### Target Management

**Conservative:**
- Target 1: 1:2 R:R
- Close full position

**Aggressive:**
- Target 1: 1:2 (close 50%)
- Target 2: 1:3 (close 25%)
- Target 3: Trail stop (25%)
