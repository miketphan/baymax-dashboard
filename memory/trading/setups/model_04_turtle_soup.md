## üéØ What is Turtle Soup?

Turtle Soup is a reversal pattern where:
1. Price breaks a key level (EQH, EQL, support, resistance)
2. Triggers stop losses
3. Immediately reverses
4. Closes back inside the range

**ICT's Definition:**
> "The Turtle Soup setup captures the liquidity sweep - the false breakout that traps breakout traders."

## üìä Long Setup (EQL Sweep)

### Step 1: Find Equal Lows
```
Look for:
‚ñ° Two or more lows at same level
‚ñ° Clear horizontal support
‚ñ° Traders have stops below
‚ñ° Liquidity sitting there

Example:
Low 1: 1.0800 (Monday)
Low 2: 1.0800 (Tuesday)
Low 3: 1.0798 (Wednesday - almost equal)

This is EQL (Equal Lows)
```

### Step 2: Wait for Break
```
Watch for:
‚ñ° Price approaching 1.0800
‚ñ° Breaking below 1.0800
‚ñ° Wick forms below
‚ñ° Quick rejection

Must have:
‚ñ° Clear break
‚ñ° Immediate rejection
‚ñ° Wick visible
‚ñ° No hesitation
```

### Step 3: Confirm Reclaim
```
Critical:
‚ñ° Price closes back above EQL
‚ñ° Within 1-2 candles max
‚ñ° Bullish candle forms
‚ñ° No further selling

Invalid if:
‚ñ° Price stays below
‚ñ° Multiple candles below
‚ñ° No clear rejection
‚ñ° Continues lower
```

### Step 4: Enter Long
```
Entry: 1.0805 (above EQL)
Stop: 1.0785 (below sweep wick)
Target 1: 1.0850 (next resistance)
Target 2: 1.0880 (extended)

Risk: 20 pips
Reward: 45-75 pips
R:R = 1:2.25 to 1:3.75
```

## üìä Short Setup (EQH Sweep)

Reverse all directions:

1. **Find Equal Highs** at resistance
2. **Wait for break** above highs
3. **Confirm reclaim** below EQH
4. **Enter short**
5. **Stop above** sweep wick
6. **Target** next support

## ‚úÖ Complete Checklist

### Setup Quality
- [ ] Clear EQH or EQL (2+ touches)
- [ ] Horizontal level visible
- [ ] Liquidity obvious (stops clustered)
- [ ] At logical support/resistance

### Sweep Phase
- [ ] Price breaks the level
- [ ] Wick extends beyond level
- [ ] Immediate rejection
- [ ] No continuation

### Reclaim Phase
- [ ] Price reclaims level quickly
- [ ] Close back inside range
- [ ] Reversal candle forms
- [ ] Structure supports reversal

### Entry Quality
- [ ] Entry above/below swept level
- [ ] Stop beyond sweep wick
- [ ] Target is opposing liquidity
- [ ] R:R minimum 1:2

### Timing
- [ ] During killzone (preferably)
- [ ] Not in chop
- [ ] Volume present on sweep

**8+ checks = Valid Turtle Soup**

## üìà Trade Management

### Entry Timing

**Aggressive:**
- Enter on close back above/below
- Better price
- Higher risk of failure

**Conservative:**
- Wait for next candle confirmation
- Worse price
- Higher win rate

### Stop Placement

**Standard:**
- Below sweep wick (longs)
- Above sweep wick (shorts)
- 5-10 pip buffer

**Tight:**
- Below EQL (if close)
- Higher risk of noise

### Target Management

**Conservative:**
- Target next structure level
- Close full position

**Aggressive:**
- Scale out at levels
- Trail extended move

   - Immediate rejection

   - Sweep should have volume
   - Rejection should have volume

4. **False Turtle Soup:**
   - Price sweeps
   - Doesn't reclaim
   - Keeps going
