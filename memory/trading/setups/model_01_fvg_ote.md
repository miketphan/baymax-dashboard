## üìä Long Setup Step-by-Step

### Step 1: Establish HTF Bias
```

‚ñ° Bullish structure (HH, HL)
‚ñ° Recent BOS to upside
‚ñ° No major resistance ahead
‚ñ° Trend is up

If bullish ‚Üí Proceed
If bearish ‚Üí Skip or look for shorts
```

### Step 2: Find LTF Impulse
```
15M Chart:
‚ñ° Strong bullish candle(s)
‚ñ° Clear momentum
‚ñ° Creates swing high/low
‚ñ° FVG visible

Example:
Candle 1: Bearish, closes at 1.0800
Candle 2: Strong bullish, opens 1.0802, closes 1.0840
Candle 3: Bullish continuation
Result: Bullish FVG created
```

### Step 3: Mark the FVG
```
Identify 3-candle pattern:

Candle 1 High: 1.0805
Candle 2 Low:  1.0802
Candle 2 High: 1.0840
Candle 3 Low:  1.0835

Bullish FVG:
Top:    1.0802 (Candle 2 low)
Bottom: 1.0805 (Candle 1 high)
Size:   3 pips
```

### Step 4: Calculate OTE
```
Impulse:
Low:  1.0800
High: 1.0840

50% = 1.0800 + (1.0840 - 1.0800) √ó 0.50
    = 1.0800 + 0.020
    = 1.0820

61.8% = 1.0800 + (1.0840 - 1.0800) √ó 0.618
      = 1.0800 + 0.0247
      = 1.0825

OTE Zone: 1.0820 - 1.0825
```

### Step 5: Check for Confluence
```
FVG: 1.0802 - 1.0805
OTE: 1.0820 - 1.0825

Do they overlap? NO

Wait for price to create new FVG during pullback,
or adjust OTE calculation.
```

**Better Example with Overlap:**
```
Impulse: 1.0800 ‚Üí 1.0860

FVG created: 1.0820 - 1.0830
OTE 50-62%: 1.0830 - 1.0837

Overlap: 1.0830 - 1.0830 (the 50% level)

```

### Step 6: Wait for Price to Reach Zone
```
Watch for:
‚ñ° Price pulling back from high
‚ñ° Entering OTE zone
‚ñ° Touching FVG area
‚ñ° Slowing momentum

Patience here is key!
```

### Step 7: Confirm Entry (Optional but Recommended)
```
Confirmation options:
‚ñ° Bullish candle at confluence
‚ñ° LTF structure break (BOS)
‚ñ° Pin bar/rejection wick
‚ñ° Volume spike

With confirmation: Higher probability, later entry
```

### Step 8: Execute Trade
