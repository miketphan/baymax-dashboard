### Why 50% - 62%?

**The 50% Level:**
- Halfway point of the move
- Psychological level
- Algorithms target it
- Strong support/resistance

**The 62% Level (Fibonacci):**
- Golden ratio (0.618)
- Natural mathematical level
- Where many retracements end
- Beyond this = potential reversal

**The Zone (50% - 62%):**
- Creates an entry "band"
- Not exact level = zone
- Allows for market variance
- Sweet spot for entries

## ğŸ”¢ Calculating OTE Levels

### Fibonacci Retracement Levels

Standard levels: 23.6%, 38.2%, 50%, 61.8%, 78.6%

**OTE Zone = 50% to 61.8%**

### Formula

**Bullish OTE (Long Entry):**
```
Impulse Low = A
Impulse High = B

50% = A + (B - A) Ã— 0.50
61.8% = A + (B - A) Ã— 0.618

OTE Zone = 50% to 61.8%
```

**Example:**
```
A (Low) = 1.0800
B (High) = 1.0900

50% = 1.0800 + (1.0900 - 1.0800) Ã— 0.50
    = 1.0800 + 0.0100 Ã— 0.50
    = 1.0800 + 0.0050
    = 1.0850

61.8% = 1.0800 + (1.0900 - 1.0800) Ã— 0.618
      = 1.0800 + 0.0100 Ã— 0.618
      = 1.0800 + 0.0062
      = 1.0862

OTE Zone = 1.0850 to 1.0862
```

**Bearish OTE (Short Entry):**
```
Impulse High = A
Impulse Low = B

50% = A - (A - B) Ã— 0.50
61.8% = A - (A - B) Ã— 0.618

OTE Zone = 50% to 61.8%
```

**Example:**
```
A (High) = 1.0900
B (Low) = 1.0800

50% = 1.0900 - (1.0900 - 1.0800) Ã— 0.50
    = 1.0900 - 0.0050
    = 1.0850

61.8% = 1.0900 - (1.0900 - 1.0800) Ã— 0.618
      = 1.0900 - 0.0062
      = 1.0838

OTE Zone = 1.0838 to 1.0850
```
