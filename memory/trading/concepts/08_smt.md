# 08 - SMT (Smart Money Tool)

**Using correlated assets to confirm or warn about potential price movements.**

---

## üéØ What is SMT?

SMT = Smart Money Tool (also called Smart Money Technique)

A divergence analysis between correlated assets that reveals hidden strength or weakness.

**ICT's Definition:**
> "SMT divergence shows when one asset is lying to you. When two correlated pairs should move together but don't, one is about to reverse."

---

## üìä Understanding Correlated Assets

### Positive Correlation
Assets that typically move in the same direction:
- EUR/USD and GBP/USD
- S&P 500 (ES) and Nasdaq (NQ)
- Gold (XAU/USD) and Silver (XAG/USD)
- AUD/USD and NZD/USD

### Negative Correlation
Assets that typically move opposite:
- USD/JPY and EUR/USD (indirectly)
- USD pairs often move opposite to each other

### Why Correlation Matters
```
Normal behavior:
EUR/USD goes UP ‚Üí GBP/USD goes UP
They move together (correlated)

SMT = When they DON'T move together
One is showing weakness/strength
```

### Definition

**What It Means:**
- Asset B is showing weakness
- Asset B's lower low is not confirmed
- Potential bullish reversal in Asset B

### How to Trade Bullish SMT

**Setup:**
1. Identify two positively correlated assets
2. Both making lows
3. Asset A = Higher Low (HL)
4. Asset B = Lower Low (LL)
5. Enter long on Asset B
6. Stop below Asset B's LL

**Example:**
```
Assets: EUR/USD and GBP/USD

Observation:
- EUR/USD makes HL at 1.0800
- GBP/USD makes LL at 1.2600
- This is bullish SMT

Trade:
- Long GBP/USD at 1.2600
- Stop below 1.2600
- Target: Next resistance or parity with EUR move
```

## üìä How to Use SMT in Trading

### Method 1: SMT as Confirmation

**When to Use:**
- You have a setup on Asset B
- Check Asset A for confirmation
- If SMT aligns = Higher probability

**Example:**
```
Setup on GBP/USD:
- Bullish OB at support
- Want to enter long

Check EUR/USD:
- EUR/USD already made HL
- GBP/USD making LL
- Bullish SMT confirmed

Result: Higher probability long on GBP/USD
```

### Method 2: SMT as Warning

**When to Use:**
- You have a setup on Asset B
- Check Asset A
- If SMT contradicts = Warning

**Example:**
```
Setup on GBP/USD:
- Bullish OB at support
- Want to enter long

Check EUR/USD:
- EUR/USD made LL (weakness)
- GBP/USD making HL (strength)
- Bearish SMT (contradiction)

Result: Warning! Setup may fail
        Consider skipping or reducing size
```

### Method 3: SMT as Primary Signal

**When to Use:**
- Clear SMT divergence on two assets
- Enter on the weaker one
- Target the stronger one's move

**Example:**
```
SMT Divergence identified:
- EUR/USD = HL (strong)
- GBP/USD = LL (weak)

Trade:
- Don't wait for other setup
- Enter long GBP/USD at LL
- Target: When GBP catches up to EUR
```

## üéØ SMT Setup Quality

### Strong SMT (High Probability)

| Factor | Why It Matters |
|--------|----------------|
| **Clear structure** | HL vs LL or LH vs HH obvious |
| **Same timeframe** | Both on 15M or both on 1H |
| **Recent swing** | Within last 10-20 candles |
| **Strong correlation** | Historically move together |
| **At key level** | SMT at support/resistance |

### Weak SMT (Low Probability)

| Factor | Problem |
|--------|---------|
| **Fuzzy structure** | Not clear HL vs LL |
| **Different timeframes** | One on 1H, one on 4H |
| **Old divergence** | From 50+ candles ago |
| **Weak correlation** | Assets don't typically move together |
| **Mid-range** | Not at significant level |

## üìã SMT Trading Checklist

- [ ] Two correlated assets identified
- [ ] Both at swing points (highs or lows)
- [ ] Clear divergence (HL vs LL or LH vs HH)
- [ ] Same timeframe analysis
- [ ] Recent swing (within 20 candles)
- [ ] At key level (support/resistance)
- [ ] Aligns with HTF structure
- [ ] Entry on weaker asset clear
- [ ] Stop placement beyond divergence point
- [ ] Target is parity or next liquidity

**7+ checks = Tradeable SMT**

### SMT Timeframe Hierarchy
```
Daily SMT: Major trend warning
4H SMT: Swing trade setup

15M SMT: Entry precision

```

### SMT Failure
```
When SMT doesn't work:
- Correlation breaks temporarily
- Major news event
- One asset decouples

Action: Exit when structure breaks
Don't hold hoping for correlation
```

## üí° Key SMT Insights

1. **SMT reveals hidden weakness/strength**
   - One asset shows true direction
   - Other is lagging or lying
   - Trade the divergence

2. **Correlation is key**
   - Use assets that SHOULD move together
   - EUR/GBP, ES/NQ, Gold/Silver
   - Not random pairs

3. **Timeframe alignment matters**
   - Both on same TF
   - Or HTF + LTF for precision
   - Mixed TF = confusion

4. **SMT is confirmation, not standalone**
   - Best with structure
   - Best with OTE
   - Best with killzone timing

5. **Take profit at parity**
   - When weak catches up to strong
   - Don't hold forever
   - SMT trade is complete
