# Protocols

> Standard operating protocols for all Nexus operations
> Last Updated: 2026-02-10

---

## Development Protocols

### Code Review Protocol
**Applies to:** All code changes

1. All changes must be reviewed before merging
2. Use descriptive commit messages
3. Run tests before submitting PR
4. Address all review comments

### Deployment Protocol
**Applies to:** Production deployments

1. Test in staging environment first
2. Verify database migrations
3. Check monitoring dashboards
4. Have rollback plan ready

## Communication Protocols

### Incident Response
**Severity Levels:**
- P0 - Service down, immediate response
- P1 - Major feature broken, respond within 1 hour
- P2 - Minor issue, respond within 24 hours

### Status Updates
- Daily standups at 9 AM
- Weekly retrospectives on Fridays
- Monthly planning on first Monday
